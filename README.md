# 🏥 Digital Health Record Analytic System

## 📌 Overview
A full-stack MERN application for managing patient records, appointments, medical data, and generating analytics dashboards for hospital performance.

## 🚀 Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React.js + Tailwind CSS |
| Backend | Node.js + Express.js |
| Database | MongoDB |
| Analytics | Chart.js |
| Auth | JWT Role-Based Authentication |
| Testing | Jest, Mocha |

## ⚙️ Installation Guide

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/digital-health-record-analytics.git
cd digital-health-record-analytics
```

### 2️⃣ Backend Setup
```bash
cd server
npm install
npm run dev
```

### 3️⃣ Frontend Setup
```bash
cd client
npm install
npm start
```
